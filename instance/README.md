# Meter Reading OCR System

This project detects and reads electric meter values from images using OCR (EasyOCR) and Flask.

## Features
- User Login System
- Image Upload
- Automatic Meter Reading Detection
- OCR-based Text Extraction

## Technologies Used
- Python
- Flask
- EasyOCR
- PyTorch
- HTML/CSS/JS

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
